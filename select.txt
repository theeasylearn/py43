------------------------------------------------------------------------------
Aggregate function

function that work upon numeric field and return single value is called Aggregate function. 

sum()
sum function returns the sum of given numeric field 

avg()
avg function returns the average of given numeric field 

min()
min() function return the minimum value of given field 

max()
max() function return the maximum value of given field 

count()
count() function count no of rows in table

Group By : 
Group by clause is used to calculate result for Aggregate function  based upon repetitive value  of specific field 

Group by clause is used along with Aggregate function. it returns sub total on repetitive value of the specific field.

having clause 
--------------------------------------------------------------
having clause is used only when we need to give condition on value returned by group by clause. so it means having clause is used only when we use group by clause

JOINS
-------------------------------------------------------------------------------------
SQL Join is used to fetch & display data from multiple table in single select statement

32) fetch customerNumber, customerName,city,state,country and orderNumber, orderDate, status of order, display it customerNumber wise ascending order 

select customerNumber, customerName,city,state,country from customers order by customerNumber
select orderNumber,orderDate,status from orders 

let us join them 

select c.customerNumber, customerName,city,state,country,orderNumber,orderDate,status from customers c,orders o where c.customerNumber=o.customerNumber order by c.customerNumber
