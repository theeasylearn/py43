Aggregate function

function that work upon numeric field and return single value is called Aggregate function. 

sum()
sum function returns the sum of given numeric field 

avg()
avg function returns the average of given numeric field 

min()
min() function return the minimum value of given field 

max()
max() function return the maximum value of given field 

count()
count() function count no of rows in table

15) fetch total payment amount from payments

17) fetch average payment amount from payments
SQL : select avg(amount) as average_amount from payments

Group By : 
Group by clause is used to calculate result for Aggregate function  based upon repetitive value  of specific field 

Group by clause is used along with Aggregate function. it returns sub total on repetitive value of the specific field. 

23) fetch total payments amount for each and every customers
SQL : select customerNumber, sum(amount) from payments group by customerNumber

24) fetch no of payments received by each every customers
SQL : select customerNumber,count(*) from payments group by customerNumber


25) fetch maximum payment amount received by each and every customer
SQL : select customerNumber,max(amount) from payments group by customerNumber
